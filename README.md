## 🚀 Professional Setup

### Technical Requirements
- Ruby 2.7+
- Jekyll 4.x
- Bundler

### Development Environment

1. **Clone the repository**
   ```bash
   git clone https://github.com/chenqijunvc/chenqijunvc.github.io.git
   cd chenqijunvc.github.io
   ```

2. **Install dependencies**
   ```bash
   bundle install
   ```

3. **Run locally**
   ```bash
   bundle exec jekyll serve
   ```

4. **Open in browser**
   Navigate to `http://localhost:4000`

### Professional Deployment

The site automatically deploys to GitHub Pages when changes are pushed to the `main` branch, ensuring professional availability and reliability.
